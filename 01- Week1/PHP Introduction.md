## PHP Pros and Cons
- Pros:
  - Mature.
  - Widely used.
  - Easy to learn.
  - Free & open source.
  - Freedom.
  - Multiple paradigms.
  - Different platforms/DBs/Servers.
  - Many frameworks & CMS (Content Management System).
- Cons:
  - Freedom (can cause bad code and security issues).
  - Syntax.
